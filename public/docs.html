Developer API Reference

Gunakan endpoint ini untuk mengintegrasikan pengiriman WhatsApp ke sistem Anda (PHP, Python, Laravel, dll).

🔐 Authentication

Semua API Endpoint bersifat privat. Anda harus menyertakan header berikut di setiap request HTTP (Bisa pilih salah satu cara yang paling familiar):

x-api-key: YOUR_API_KEY_HERE
atau
Authorization: Bearer YOUR_API_KEY_HERE

Dapatkan string token rahasia ini di menu utama Dashboard.

1. Send Text Message

POST/api/v1/send-message

Mengirim pesan teks biasa ke WhatsApp target.

Parameter (JSON)TypeDescription
numberStringNomor tujuan (Contoh: 08123456789 atau 628123456789)
messageStringIsi pesan teks yang ingin dikirim
{
    "number": "08123456789",
    "message": "Halo! Ini notifikasi sistem."
}

2. Send Image

POST/api/v1/send-image

Mengirim gambar dan pesan caption (Opsional).

Parameter (JSON)TypeDescription
numberStringNomor WA tujuan
image_urlStringURL dari gambar (contoh: https://domain.com/gambar.jpg)
captionString(Opsional) Teks bawaan gambar
{
    "number": "628123456789",
    "image_url": "https://images.unsplash.com/photo-1541963463532-d68292c34b19",
    "caption": "Silakan lihat brosur terlampir."
}

3. Send Document / File

POST/api/v1/send-document

Mengirim file dokumen seperti PDF, Word, atau zip.

Parameter (JSON)TypeDescription
numberStringNomor surat tujuan
document_urlStringURL dari dokumen PDF/Word yang bisa diakes publik
filenameString(Opsional) Nama yang akan tampil pada WhatsApp. Default: Document
mimetypeString(Opsional) Default: application/pdf
{
    "number": "628123456789",
    "document_url": "https://www.w3.org/WAI/ER/tests/xhtml/testfiles/resources/pdf/dummy.pdf",
    "filename": "Kwitansi-2026.pdf",
    "mimetype": "application/pdf"
}

Contoh Penggunaan cURL CLI

curl -X POST "http://localhost:3000/api/v1/send-message" \
-H "Content-Type: application/json" \
-H "Authorization: Bearer wa_d41d8cd98f00b204..." \
-d '{"number":"08123456789", "message":"Halo dari APIWA!"}'

⚡ Webhook / Inbound Messages (Callback)

INBOUND POSTDiterima oleh Aplikasi Internal Anda

Pendaftaran Webhook URL dapat dilakukan 1 click melalui Dashboard Web. Jika Anda mengisinya, APIWA Gateway akan otomatis mengirimkan data HTTP POST (Real-Time) setiap ada kiriman pesan WhatsApp masuk ke nomor bot Anda.

Struktur File JSON Webhook (Callback Payload) yang akan di-post secara asinkron ke URL Anda:

{
    "number": "628994044810",
    "name": "Alan Alan",
    "message": "Halo kak, pesanan 001 kapan dikirim?",
    "timestamp": 1729384821,
    "messageId": "3EB04DXX..."
}

Gunakan informasi payload ini pada direktori server PHP / Laravel Anda untuk menyimpan ke Database Kasir, atau dibalas balik menggunakan Endpoint `send-message` (Auto-Reply 2 Arah).
